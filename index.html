十分钟建立个人博客—实际用了两天

在网上看到通过Github Pages技术、Hexo来急速建站的方法，很有意思就试了一下，下面是过程中的一些记录和问题解决方法

###参考资料
1.主要思路：所有思路都来自于这篇博客十分钟建站。
2.Hexo主题：这篇博客的Hexo主题不错，名叫next。下面是主题相关的配置方法：
next五分钟快速安装
next第三方服务
主题内建样式

###过程问题

1.如何使用多说

next可以使用多说来实现对文章的评论，Hexo中如何使用多说参考Hexo使用多说教程，发现在网站配置文件(_config.yml)中直接配置duoshuo_shortname就可以，不需要配置后面的代码。duoshuo_shortname的获得方法见网页多说创建网站；至于其中什么是多说的duoshuo_shortname，十分钟建站中有详细的描述。

2.无法将程序完整提交到Github

在本地测试完成后，出现了无法完整提交代码到Github的情况，发现其中的next主题是通过git clone下载的，在提交时，也提交了next，而next我不是contributor，所以无法提交。解决的办法就是在next的github网页下载稳定版代码的zip包，解压，放入项目。清空Github中的所有代码，重新提交后成功。

3.Hexo配置完成后无法部署

提交代码后，进入本地项目根目录，输入Hexo g生成静态页面，Hexo d完成最终部署。在部署过程中出现ERROR Deployer not found: git错误，通过 npm install hexo-deployer-git –save最终解决该问题。

最后，还可以给自己的网站添加一个好的域名，我比较懒就没有弄。另外，十分钟建站最后的几篇参考文章可以以后细看，如：
Hexo 静态博客使用指南介绍了Hexo的安装，配置，部署和更新方法。如何搭建一个独立博客——简明 Github Pages与 jekyll 教程介绍了Github Pages技术的使用方法。动动手指，不限于NexT主题的Hexo优化（SEO篇）介绍了Next主题的优化技术。
